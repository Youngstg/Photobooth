TEMPLATE FRAME PHOTOBOOTH
=========================

Folder ini berisi semua template frame yang telah di-generate.

Total: 48 template (12 tema × 4 variasi jumlah foto)

CARA GENERATE TEMPLATE:
1. Buka file: d:\Download\Photobooth\generate-templates.html di browser
2. Halaman akan otomatis generate semua template
3. Klik tombol "Download All as ZIP" untuk download
4. File akan terdownload satu per satu ke folder Downloads
5. Pindahkan semua file PNG ke folder ini
